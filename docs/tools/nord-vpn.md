# Nord VPN

![Screenshot](../img/tools/nord-vpn.png)

## Installation

### Download Nord VPN

``` sh
sudo wget -qnc https://repo.nordvpn.com/deb/nordvpn/debian/pool/main/nordvpn-release_1.0.0_all.deb
```
### Install Nord VPN

``` sh
sudo apt update
sudo apt install -y nordvpn
```

### Setup Nord VPN

#### Login

``` sh
nordvpn login
```
